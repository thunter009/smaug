# MapToPoster

Transform favorite cities into beautiful, minimalist map poster designs. Create and export visually striking posters with code.

## Key Features
- Generate posters for any city worldwide
- 17 different visual themes (sunset, blueprint, neon_cyberpunk, japanese_ink, terracotta, etc)
- Built on OpenStreetMap data via OSMNX and matplotlib
- Customizable dimensions and distance radius
- Export to various resolutions (Instagram, mobile wallpaper, HD/4K, A4 print)
