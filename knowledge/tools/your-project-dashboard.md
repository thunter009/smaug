# Your Project Dashboard

Open-source app that scans your dev folder for git repos and provides a unified project dashboard.

## What it does
- Auto-discovers git repositories in your development folder
- Analyzes recent activity and project state
- Shows what you've built and what you're currently working on
- Provides at-a-glance project status overview
