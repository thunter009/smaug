# Clauntty

iOS Terminal with native Claude Code support, built for iPad productivity.

**Repo:** https://github.com/eriklangille/clauntty
**Language:** Swift
**Stars:** 27

## Overview

Clauntty is a mobile terminal for iOS that forks Ghostty and brings Claude Code support to iPad. Built over Christmas by Erik Langille, it provides native integration with Claude Code for on-the-go coding and server management.

## Features

- Native Claude Code integration
- Ghostty terminal under the hood
- SSH client capabilities
- TestFlight available
- Open source

## Use Cases

- iPad coding environment
- Remote server management
- Quick SSH access while mobile
- Agent monitoring on the go
