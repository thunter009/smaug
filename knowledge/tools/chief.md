# Chief

## Overview
CLI wrapper for Claude Code that automates project development by breaking work into tasks and running Claude Code in a loop.

## Key Features
- Wraps Claude Code in task-based loop
- Auto-commits after each task
- Zero configuration required
- Single binary deployment
- Pretty TUI
- Picks up where it left off

## Usage
Define a PRD, run `chief`, then do literally anything else. System works through tasks autonomously and commits progress.
