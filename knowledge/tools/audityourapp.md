# AuditYourApp - Supabase & Firebase Security Scanner

Enterprise-grade security scanner that detects vulnerabilities in Supabase and Firebase applications.

## What it does

Automates security audits of web and mobile apps for:
- RLS (Row Level Security) policy misconfigurations
- Exposed storage buckets
- Leaked API keys
- Other common backend vulnerabilities

## Key features

- No setup required - paste and scan
- Web + mobile app support
- Automated pentest workflow

Created by security researcher Burak Eregar after auditing 15 "vibecoded" apps and discovering 600k+ users' data exposed due to misconfigured backends. Tool prevents apps from ending up on hacker forums.
