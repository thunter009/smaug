Google Toolbox provides Google's own diagnostic tools for DNS, email, headers, and network issues. Surprisingly useful for debugging real-world infrastructure problems.

A collection of network and email diagnostics tools maintained by Google that solve actual production issues.

## Key Features

- **DNS Diagnostics** - Troubleshoot DNS propagation and configuration
- **Email Header Analysis** - Debug email delivery and authentication issues
- **Check MX** - Verify mail exchanger records
- **Check CNAME** - Check canonical name records
- **SPF/DKIM/DMARC Tools** - Email authentication validation
- **Network Troubleshooting** - Various network diagnostic utilities

## Use Cases

- Debugging email delivery issues
- DNS propagation troubleshooting
- Email authentication setup verification
- Network configuration validation
- Production issue diagnosis
