JSON Crack transforms messy JSON into clean visual trees. Makes understanding large APIs and configuration files dramatically easier than staring at raw text.

A visual debugger and analyzer for JSON data that instantly converts unstructured data into explorable, interactive diagrams.

## Key Features

- **Visual Tree Representation** - See JSON structure at a glance
- **Interactive Navigation** - Explore nested data easily
- **Large File Support** - Handle complex APIs and configs
- **Copy/Paste Ready** - Paste any JSON and get instant visualization
- **Shareable Diagrams** - Export and share visualizations

## Use Cases

- Debugging API responses
- Understanding complex JSON structures
- Exploring configuration files
- Learning JSON structure visualization
- API response analysis
