# Happy - Claude Code Mobile Client

## Overview
Mobile and web client for Claude Code and Codex with E2E encryption, voice support, and cross-device session management.

## Key Features
- **E2E Encryption:** Secure wrapper around Claude Code/Codex
- **Voice Support:** Realtime voice interface
- **Cross-Device:** Seamless session switching between desktop and mobile
- **Smart Control:** Press keyboard to switch control back to desktop instantly

## Usage
```bash
# On desktop instead of claude
happy

# Or with codex
happy codex
```

On mobile, session restarts in remote mode. Keyboard input on desktop instantly switches control back.
