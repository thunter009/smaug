# Textream

## Overview
Lightweight macOS teleprompter app using on-device speech recognition to highlight scripts as you speak in real-time.

## Key Features
- On-device speech recognition (no cloud)
- Dynamic Island-style overlay
- Works offline
- Tiny footprint: 913kb only
- Real-time script highlighting as you speak

## Use Cases
- Live streaming
- Interviews
- Presentations
- Podcasts

## Installation
```bash
brew install f/textream/textream
```

## Tech Details
- macOS app
- No cloud dependency
- Minimal resource usage
