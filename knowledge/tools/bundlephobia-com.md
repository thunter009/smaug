Bundle Phobia shows how big an npm package really is before you install it. Helps you avoid bloating your app with "tiny" libraries that aren't actually that tiny.

A package size analyzer that prevents unexpected bloat by showing gzip and minified sizes of npm packages before installation.

## Key Features

- **Size Analysis** - See minified and gzip compressed sizes
- **Dependency Tree** - Understand size impact of dependencies
- **Search** - Quick lookup for any npm package
- **Size History** - Track how package size changes over versions
- **Comparisons** - Compare similar packages by size

## Use Cases

- Evaluating npm packages before adding to project
- Optimizing bundle size
- Finding smaller alternatives to heavy packages
- Understanding dependency weight
- Performance-conscious development
