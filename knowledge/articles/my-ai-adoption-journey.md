# My AI Adoption Journey

**Author:** Mitchell Hashimoto (creator of Terraform)

**URL:** https://mitchellh.com/writing/my-ai-adoption-journey

## Overview
A practical, step-by-step guide on setting up and using a companion coding/research agent running locally as a developer. Addresses how to integrate AI tools into development workflow without getting overloaded or anxious.

## Key Topics
- Setting up local AI companions for coding/research
- Managing AI tool adoption
- Avoiding information overload and anxiety
- Practical implementation steps
- Workflow integration

## Target Audience
Developers interested in adopting AI coding/research agents without experiencing productivity anxiety or overwhelm.
