# Echo - iOS SSH Client

Fast, simple SSH client for iPad that runs Ghostty under the hood.

**Link:** https://replay.software/echo
**Platform:** iOS App Store
**Creator:** almonk

## Overview

Echo is a new SSH client for iOS that transforms iPad into a productive coding environment. Built on Ghostty, it enables remote server management, agent monitoring, and terminal-based work on mobile.

## Features

- Fast and simple SSH client
- Ghostty terminal backend
- Agent management on iPad
- OpenClaws monitoring
- Server management capabilities
- Full terminal access (top, etc.)
